# Catan Tournament Tracker - Official Scoring Rules

This document outlines the official scoring rules implemented in the Catan Tournament Tracker, based on official Catan rulebooks and tournament guidelines.

## üìä Victory Point Requirements by Game Type

### Base Catan (3-4 Players)
- **Victory Points to Win:** 10 VP
- **Description:** Classic Settlers of Catan gameplay

### Base Catan (5-6 Players)
- **Victory Points to Win:** 10 VP
- **Description:** Extended base game with 5-6 player extension
- **Note:** Same VP requirement as 3-4 player game

### Seafarers Expansion
- **Victory Points to Win:** Varies by scenario (typically 10-14 VP)
- **Default:** 10 VP
- **Description:** Adds ships, islands, and exploration mechanics

### Cities & Knights Expansion
- **Victory Points to Win:** 13 VP
- **Description:** Adds city improvements, knights, and barbarian invasions
- **Key Difference:** Requires 3 additional VP compared to base game

### Seafarers + Cities & Knights Combined
- **Victory Points to Win:** Typically 15 VP
- **Description:** When playing both expansions together
- **Calculation:** Seafarers scenario VP requirement + 2 VP for C&K

### Traders & Barbarians
- **Victory Points to Win:** Varies by scenario (typically 10 VP)
- **Description:** Collection of variant scenarios

### Explorers & Pirates
- **Victory Points to Win:** Varies by scenario (typically 10 VP)
- **Description:** Adds missions, spice trading, and pirate ships

## üèÜ Tournament Scoring Rules

Based on official Catan tournament guidelines:

### Game Winner Determination
1. First player to reach the required VP on **their turn** wins
2. Players cannot win on another player's turn (even if they reach required VP)
3. VP development cards can be revealed immediately to win if they bring you to required VP

### Tournament Ranking (Preliminary Rounds)

Players are ranked in the following order:

1. **Number of Victories** (primary ranking)
   - Most game wins ranks highest

2. **Total Victory Points** (first tiebreaker)
   - Sum of all VP across all games
   - Maximum 10 VP per game counted (even if player scored more)

3. **Victory Point Percentage (VP%)** (second tiebreaker)
   - For each game: Player's VP √∑ Total table VP = Game VP%
   - Sum all game VP% for total VP%
   - For 3-player games: Add "dummy score" (average of 3 players) to normalize

4. **Higher Placement Finishes** (third tiebreaker)
   - Number of 2nd place finishes
   - Then number of 3rd place finishes

5. **Drawing Lots** (final tiebreaker)
   - Random selection if all else is tied

### Important Tournament Notes
- Games continue until someone reaches required VP (no time limits on games)
- Individual turns may have time limits
- Development cards with VP can be played immediately for victory
- Knight cards must wait until next turn (cannot win immediately after playing)

## üìù Using the Tracker

### Recording Scores
1. Select the appropriate game variant
2. The tracker will show relevant VP sources for that variant
3. Enter the actual VP earned from each source
4. Total VP is calculated automatically
5. Players are ranked by total VP

### Best Practices
- Record all VP sources accurately
- Double-check totals match the game winner's declared VP
- For tournament play, note that max 10 VP per game for ranking calculations
- Use the "description" field to note any special scenario rules
